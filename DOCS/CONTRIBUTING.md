# 🤝 Como Contribuir

Ficamos felizes com seu interesse em melhorar o AdvogaFácil!

### Como enviar melhorias?
1. Faça um **Fork** do projeto.
2. Crie uma **Branch** para sua funcionalidade (`git checkout -b feature/NovaIntegracaoTJPA`).
3. Faça o **Commit** de suas alterações seguindo o padrão [Conventional Commits](https://www.conventionalcommits.org/).
4. Envie para o **Push** para a sua branch (`git push origin feature/NovaIntegracaoTJPA`).
5. Abra um **Pull Request**.

### Regras de Código
- Siga as normas da PEP 8 para código Python.
- Sempre adicione testes unitários para novas integrações de scrapers.
- Documente novas funções no `docs/`.
